# Glossary

- x402: 결제 필요 리소스 접근을 위한 HTTP challenge 기반 패턴
- Invoice: 서버가 발행한 결제 청구 원문
- Verification: 모바일 환경에서 사용자 승인 및 서명 검증
- Settlement Proof: 결제 완료를 증명하는 온체인/외부 증빙
- SSOT: Single Source of Truth
- TTM: Transaction Terms Manifest, 거래 조건 표준 문서
- CR: Consent Receipt, 사용자 동의 증빙
- SR: Settlement Receipt, 정산 결과 증빙
- Atomic Amount: 통화/자산 최소 단위 정수(BigInt) 표현
- JCS: JSON Canonicalization Scheme (RFC 8785)
- Delegation First: 불변식은 플랫폼 고정, 운영값은 개발자/운영팀 위임 원칙
- Policy ID: 운영 파라미터를 외부 정책 레지스트리에서 주입하기 위한 참조 키
